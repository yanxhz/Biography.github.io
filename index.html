WELCOME TO MY BIOGRAPHY

ABOUT

Nama saya adalah Adrian Michael Duha. Saya dilahirkan pada 09 Oktober 2021 di Gunungsitoli. Orang tua saya bernama Adventinus Duha dan Yenniawati Hulu. Saya merupakan anak kedua dari tiga bersaudara. Kakak perempuan saya bernama Christina Eben Hezerin Duha, ia sedang menumpuh kuliah di salah satu Universitas di Solo, dan Adik laki laki saya bernama Rizki Mo'arota Duha yang sedang menempuh Sekolah Menegah Atas.

Pendidkan

Orang tua menyekolahkan saya di TK CENDRAWASI KOTA GUNUNGSITOLI, SD SWASTA RK MUTIARA KOTA GUNUNGSITOLI, SMP NEGERI 1 KOTA GUNUNGSITOLI, SMA NEGERI 1 KOTA GUNUNGSITOLI dan Sedang menumpuh kuliah swasta di luar pulau tepatnya di Kota Medan. Saya kuliah di Universitas Mikroskil mengambil jurusan Teknik Informatika. Saya mengambil jurusan tersebut karena saya orangnya suka bermain game dan sangat ingin megetahui teknologi.

Kegiatan Semasa Kuliah

Semasa kuliah, saya menjabat sebagai Kepala Divisi MLBB di UKM Mikroskil Esports selama Setahun lebih hingga sekarang . Tugas saya selama menjadi Kepala Divisi MLBB yaitu mengkoordinasikan tournament MLBB setiap minggunya, dan di divisi tersebut memiliki 90+ anggota aktif. Selain itu saya juga sempat mengikuti Program Kampus Merdeka yaitu Study Independen di Dicoding tentang Pengembangan Front-End Web dan Back-End selama enam bulan. Pada program ini saya mempelajari banyak hal mulai dari Memulai Dasar Pemrograman untuk Menjadi Pengembang Software , Pengenalan ke Logika Pemrograman (Programming Logic 101), Belajar Dasar Git dengan GitHub, Belajar Dasar Pemrograman Web, Belajar Membuat Front-End Web untuk Pemula, Belajar Fundamental Front-End Web Development, Menjadi Front-End Web Developer Expert, Cloud Practitioner Essentials (Belajar Dasar AWS Cloud), Belajar Dasar Pemrograman JavaScript, Belajar Membuat Aplikasi Back-End untuk Pemula dan mengikuti setiap meet tentang Soft Skill yang dibawakan oleh mentor - mentor expert.
